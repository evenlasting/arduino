# 小学期作业

一个自己玩别踩白块的机器
<br/>
一个能告诉你，你别踩白块最多能玩多少分的机器
<br/>
一个超傻的别踩白块游戏

---------------------------------


* 优点
  * 改进机械结构，增加极限分数
  * 增加与玩家的交互
  * 可读取游戏速度
* 缺点
  * 双线程的arduino设计很难把控延时
  * 所有的问题几乎都是来源于双线程，头疼
